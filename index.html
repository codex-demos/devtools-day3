Debugging Exercise: Capture Secret Messages Part 2

Objective

Use Google Developer Tools to capture secret messages from a deployed website. This exercise is designed to help you understand and utilize breakpoints in JavaScript debugging.

Instructions

  1. Open Google Developer Tools (right-click on the page and select "Inspect" or use `Ctrl+Shift+I`/`Cmd+Option+I`).
  2. Navigate to the "Sources" tab in the Developer Tools.
  3. Find the script.js file in the file explorer within the "Sources" tab.
  4. Set breakpoints at the lines where fetchSecretMessage function calls are made.
  5. Interact with the website as needed to trigger the JavaScript code execution.
  6. When each breakpoint is hit, capture the secret message displayed in the source local scope window.
  7. There are 10 secret messages corresponding to tasks 11-20.
  8. You may have to skip over errors to continue seeing output

Assignment Submission

Create a GitHub repository with a README.md file. In the README, list all 10 secret messages along with their corresponding task numbers. Submit the link to your GitHub repository.
